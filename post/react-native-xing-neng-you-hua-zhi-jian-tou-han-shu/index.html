react native 性能优化之箭头函数

2019-11-12

关于箭头函数的写法
onPress={this.show("XX")}
当react native 解析模板时就会执行。会出现执行很多次。
onPress={this.show.bind(this,"XX")}
当页面的state改变时会导致页面再次刷新，多次执行绑定事件。
onPress={()=>this.show("XX")}
方法需要的时候执行
onPress={this.show}
